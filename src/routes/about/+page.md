<h1>关于中国 PWA 开发者日</h1>

<h2>为什么参加？</h2>

中国 PWA 开发者日的使命是助您使用渐进式 Web 应用（PWA）取得成功。

创建此活动是为了分享国内和国际厂商的 PWA 的成功案例，传播有关 PWA 的技术及实践，包括如何成功构建 PWA 以及将 PWA 技术集成到您现有的 Web 应用中。我们不专注于推销特定的 Web 框架或技术栈。

该活动免费。

<h2>主办方</h2>
中国 PWA 开发者日由参与创建 PWA 技术的几家公司的人员合作举办。
